# Benchmark methodology

The product claims a recommended balance of performance, image quality and combat visibility; it does not claim universally optimal settings or a guaranteed FPS gain. The large current-FPS value is live while the Graphics page is open. The eight-second before/after `GetFramerate()` comparison is a convenient local estimate, not a replacement for the controlled benchmark below.

For each profile, test entry 1080p, mainstream 1080p, mainstream 1440p, high-end 1440p and 4K systems. Record CPU, GPU, RAM, OS, resolution, Render Scale, driver, WoW build, addon version, average FPS, minimum observed FPS, frametimes where available, visual observations and mechanic visibility. Repeat indoor low-load, current capital, current open world, Mythic+, raid and battleground scenarios after a warm-up run. Use the same character, route, encounter, graphics API, foreground/background limits and display mode for comparisons.

Do not change monitor, resolution, refresh rate, graphics API, HDR, latency settings, V-Sync or FPS limits as part of an addon benchmark. Render Scale remains unchanged unless reliable in-client resolution detection establishes a resolution at or below 2560×1440; that decision remains intentionally disabled in v1.0.0 pending live validation.
