# S-Tier Blizz Settings

Лёгкий аддон WoW Retail для применения проверенных встроенных настроек Blizzard. Он не заменяет интерфейс Blizzard и не требует ElvUI, WeakAuras или других UI-аддонов.

Есть два независимых модуля: **Графика** и **Интерфейс и игра**. Режимы графики: **Единые настройки везде** и **Оптимизация рейдов**. Каждое применение создаёт резервную копию; поддерживаются сохранение личного профиля, безопасный экспорт `STBS1:` и импорт без исполнения кода.

Откройте окно командой `/stier` или `/stbs`. Доступны: `graphics`, `interface`, `apply`, `save`, `export`, `import`, `backup`, `restore`, `debug`.

Аддон сохраняет разрешение, устройство/монитор, частоту, V-Sync, FPS-лимиты, режимы задержки, звук, доступность, бинды, масштаб UI, мышь и другие личные параметры. В нём нет телеметрии, сети, рекламы, платных функций и призывов к донатам.

Базовая совместимость: Retail 12.0.7, Interface 120007 (исходный код Blizzard UI build 68453). Перед использованием в серьёзном контенте проверьте аддон в живом клиенте.

S-Tier Blizz Settings — независимый общественный аддон, не связанный с Blizzard Entertainment и не одобренный ею.
